# ⚡ CodeDocGen: AI-Powered Technical Intelligence

> **"Turning Complex Code into Human Clarity."**

CodeDocGen is a high-performance documentation suite designed for modern engineering teams. It bridges the gap between raw source code and professional technical reports by using heuristic AI to analyze logic intent and behavioral patterns.

---

## 🌟 Key Features

### 🧠 Heuristic AI Explanation Engine
The system doesn't just read code; it understands **intent**. It automatically detects authentication blocks, data persistence handlers, and validation logic to provide human-readable behavioral summaries.

### 📦 Bulk Project Ingestion
- **ZIP Support**: Upload entire repositories in `.zip` format for instant project-wide documentation.
- **Batch Processing**: Handle multiple files simultaneously with a centralized analytics dashboard.

### 📊 Advanced Developer Analytics
Integrated **Chart.js** visualizations show your project's technology distribution (JS vs. Python) and code complexity metrics in real-time.

### 🖥️ Professional IDE Code Viewer
View your source code in a specialized editor container featuring:
- **Atom One Dark** syntax highlighting via `highlight.js`.
- **macOS Style** window controls and premium typography.
- **Fira Code** monospaced font integration.

### 📄 Multi-Format Exports
- **Premium PDFs**: High-fidelity technical reports for client handover.
- **Markdown (MD)**: Clean exports for GitHub/GitLab READMEs.

---

## 🛠️ Technical Stack

- **Backend**: Node.js & Express.js
- **Database**: MySQL (Enterprise Grade)
- **Engine**: Custom Heuristic Regex Parser
- **UI/UX**: Vanilla CSS (Luxury Glassmorphism) & Bootstrap 5
- **Export**: PDFKit & FileSystem Streams
- **Charts**: Chart.js 4.0
